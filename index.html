四人共同段：7 月 12 日下午到 7 月 14 日朋友返程前。
已订住宿：黔程时光酒店(观山湖公园林城西路地铁站店)，7 月 12 日-14 日两晚。
主线：贵州抹茶、贵阳咖啡、老城吃喝、桃源河漂流；7 月 14 日后再决定黄果树或花溪 / 青岩。
不安排博物馆；活动 / 演出不作为行程取舍依据。喜茶和商场只作为雨天、等人、避暑、拖行李时的补充。

7/14 周二 朋友返程，你们继续 返程早走老城早午餐；返程晚走青岩 / 夜郎谷 后两晚建议换到喷水池 / 大十字 / 甲秀楼 / 太平路

二、每天怎么走

7/12 周日：抹茶 + 咖啡 + 老城夜游

时间 安排 说明
07:54-11:34 G3722 佛山西 → 贵阳东 到站后直接打车去酒店
11:40-14:00 贵阳东 → 黔程时光酒店 → 寄存 / 入住 + 简单午餐 午餐别吃太撑，下午要喝抹茶和咖啡
14:30-15:20 去茶山（南国花锦店） 贵州本土茶饮，适合作为抹茶第一站
15:40-16:30 青绿智游·抹山序（北京路方向） 抹茶主题更强，出发前用高德确认营业
16:45-17:45 咖啡 1 家：乔治队长 / 对的咖啡 / QUANTIC 不要一天喝太多，后两天还能补
18:00-20:30 酸汤鱼 / 豆米火锅 → 甲秀楼 / 太平路 第一晚重点是老城氛围和正餐
20:30 后 青云市集夜宵 冰浆、豆腐圆子、烤肉、小吃；不要当唯一正餐

雨天改法：把户外步行缩短，集中在南国花锦 / 喷水池 / 国贸一带解决抹茶、正餐和等位。

7/13 周一：桃源河漂流

时间 安排 说明
08:30-09:00 酒店出发 四人建议打车或包车，少换乘
10:30-14:30 桃源河漂流 检票和水位以景区当天公告为准
14:30-16:00 洗漱换衣 + 回贵阳 带干衣服、毛巾、防水袋、涉水鞋
18:00 后 贵厨·酸汤牛肉 / 黔大毕烙锅 / 新大新豆米火锅 漂流后优先热汤或烙锅，不再安排重步行

雷雨或景区临时调整：改天河潭半日、阿云朵仓、老城吃喝，别把全天都压在水上项目上。

三、吃喝优先清单

抹茶 / 茶饮

优先级 店名 推荐点 放进行程
A 去茶山（南国花锦店） 贵州本土茶饮，适合点铜仁抹茶相关饮品、本土水果茶 7/12 第一站
A 青绿智游·抹山序 抹茶主题强，有饮品和甜点 7/12 第二站，出发前确认营业
B 生根抹茶行 小众抹茶体验，适合老城走累后坐一会儿 喷水池 / 民生路 / 蔡家街顺路时去
B 吾茶白·贵茶潮饮烘焙概念店 偏贵茶品牌和抹茶烘焙 抹山序或去茶山没安排上时
C 喜茶（南国花锦 / CCPARK / 国贸 / 花果园） 只负责等人、避雨、补给 不作为主线

正餐 / 早餐

场景 优先店名 推荐点
第一晚酸汤 老凯俚酸汤鱼（省府店） 酸汤鱼、辣子鸡、红糖糍粑；适合第一次正式吃贵州酸汤
7/12 顺路火锅 新大新豆米火锅（南国花锦店） 和去茶山同商圈，豆米火锅、酥肉、豆腐圆子
7/13 漂流后 贵厨·酸汤牛肉（花果园国际中心店） 酸汤牛肉适合漂流后补体力，四人分食方便
7/13 低体力夜宵 黔大毕烙锅 烙锅、蘸水，离观山湖方向合理
7/14 轻午餐 杨姨妈丝娃娃（飞山街店） 丝娃娃、洋芋粑粑，口味比火锅轻

7/14 早餐 翁恒志老贵阳全牛粉（尚义路） 比肠旺面更容易被多数人接受
酒店附近早餐 彝大锅·砂锅羊肉粉 热汤、羊肉、砂锅香，适合快速早餐

五、雨天、预约和交通

雨天调整

情况 走法
7/12 下雨 酒店 → 南国花锦去茶山 + 新大新豆米火锅 → 室内咖啡 / 喜茶等位 → 雨停再去甲秀楼
7/13 漂流受影响 先看景区公告；改天河潭 / 阿云朵仓 / 老城吃喝
7/14 朋友拖行李 喷水池 / 大十字吃饭 → 国贸 / 南国花锦 / 花果园等车
7/15 黄果树不稳 改花溪牛肉粉 + 青岩 / 夜郎谷，或市区咖啡吃喝

预约和风险

项目 必看
桃源河漂流 景区公告、天气预警、票种、退改规则、干衣服和防水袋
黄果树 门票、水帘洞预约、身份证、大扶梯是否需要、返程交通
抹山序 / 小众咖啡 出发前用高德确认营业状态
正餐 高峰提前看排队；链条店按最近分店走，不执着总店

六、出发前检查清单

日期 必查项 不满足时怎么改
7/12 高铁是否准点、酒店能否寄存、去茶山 / 抹山序营业、晚餐排队 到得晚就删掉 1 家咖啡；下雨就先吃正餐
7/13 桃源河公告、雷雨预警、票种、回程车程 强雷雨改天河潭 / 阿云朵仓 / 老城吃喝
7/14 朋友车次 / 航班、行李寄存、送站时间、是否换酒店 返程早只吃早餐；返程晚再考虑青岩 / 夜郎谷
7/15 黄果树门票、水帘洞预约、身份证、天气、返程交通 水帘洞约不上就改花溪线或市区线
7/16 退房时间、行李寄存、返程票、最后一餐位置 只保留市中心吃喝，不再加远点

随身准备：漂流日带手机防水袋、毛巾、干衣服、涉水鞋或拖鞋、眼镜固定绳；黄果树日带身份证、防滑鞋、雨具、充电宝；每天带纸巾、肠胃药和防晒。
